## Usage

Open any `tiaa-style-guide-YYYY-MM-DD.html` file directly in a browser — it is self-contained (no build step, no server needed). Or deploy to the VPS as a static file.

## Updating

1. Export a new Elementor Kit from WordPress admin → Elementor → Tools → Export Kit
2. Upload the zip to Claude with this prompt: *"Generate an updated style guide from this kit export"*
3. Save the output as `tiaa-style-guide-YYYY-MM-DD.html` and commit to this directory
4. Keep prior versions for reference
