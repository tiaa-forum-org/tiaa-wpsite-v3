06 — Notes
For Maintainers

Key rules for keeping the design system consistent.

Where to change colors

Elementor → Site Settings → Global Colors. Changes here propagate everywhere the global is referenced. Never hardcode hex values in widgets if a global color exists.

Where to change fonts

Elementor → Site Settings → Global Fonts. Set responsive sizes here (desktop/tablet/mobile), not per-widget. Only use per-widget overrides for one-off exceptions.

Where shared CSS lives

tiaa-elementor plugin → assets/css/tiaa-elementor.css. If a class or rule is used on more than one element or page, it belongs here — not in Elementor page CSS or widget CSS.

Content width standards

Single-column 760px
Two-column 960px
Full layout max 1200px
Section padding 80px top/bottom
