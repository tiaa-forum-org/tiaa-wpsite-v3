# TIAA WP Site V3

## Purpose
This repository contains documentation, configuration exports, and assets for the TIAA Forum  WordPress site which is envisioned as a reference and a demonstration of the architecture and development practices used by the TIAA Admin team for the development of the site. Work on this site is began early in 2026 and is ongoing.

The other main purpose of the site is to serve as an interface to the main purpose of tiaa-forum.org which is to offer to the larger AA community a place to connect, learn, and engage with each other (share expeience, strength, hope, and knowledge) inn the support of the AA primary mission of "Carrying the message...".
The site is built using:
- WordPress
- Elementor Pro (Presentation Layer)
- (list of plugins, etc.)

The forum community per se (discourse.tiaa-forum.org) is hosted on a seperate Discourse instance managed as a managed Discourse service by CDMYK inc.

## Design Philosophy

This project prioritizes:

- Long-term maintainability
- Volunteer-safe editing
- Documentation-first architecture
- Separation of Content, Presentation, and Logic
- Git as the source of truth for structure and code
- Production database as source of truth for content

## Repo Scope

This repo may contain:

- Documentation

- Deployment notes

## Non-Goals

This repo does NOT attempt to:

- Store full WordPress database snapshots
- Store full uploads directory
- Replace production backup systems
